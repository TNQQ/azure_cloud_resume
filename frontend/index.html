Azure Cloud Resume Challenge

  • Beschreibung:

    • Im Rahmen der Cloud Resume Challenge habe ich eine voll funktionsfähige und sichere Lebenslauf-Website entwickelt, die auf Azure und modernen Webtechnologien basiert. Die statische Seite wurde mit HTML und CSS erstellt und auf Azure Blob Storage bereitgestellt, wobei Azure CDN für die schnelle und zuverlässige Inhaltsauslieferung sorgt. Für die sichere Kommunikation wurde HTTPS implementiert, und die Domain-Registrierung erfolgte über Azure DNS. Ein Besucherzähler, der in JavaScript eingebunden ist, verbindet sich mit Azure Functions zur Backend-Verarbeitung und nutzt Azure Cosmos DB zur Speicherung der Daten. CI/CD-Pipelines wurden mithilfe von GitHub Actions für eine automatisierte Bereitstellung eingerichtet.

    Verwendete Technologien:

  • Frontend: HTML, CSS, JavaScript
  • Backend: Azure Functions, Azure Cosmos DB,JavaScript
  • Infrastruktur: Azure Blob Storage, Azure CDN, Azure DNS
  • Beobachtbarkeit: Azure Monitor, Application Insights
  • DevOps: Git, GitHub, GitHub Actions
